IoT Weather Dashboard with Machine Learning

Predicting temperature using real-time weather data and regression modeling

Overview

This project combines real-time weather data collection with machine learning to build a temperature prediction system. Using historical data retrieved from the OpenWeatherMap API, a regression model was trained to forecast upcoming temperature values based on features like humidity, wind speed, cloud cover, and pressure. A Flask-based web dashboard presents the model’s predictions, evaluation metrics, and recent weather trends in a clean, accessible interface.

Images & Screenshots

Key Features

  • Fetches recent weather data from OpenWeatherMap's API
  • Trains a regression model (Linear Regression / XGBoost) with over 97% R² accuracy
  • Predicts future temperature values for specified cities
  • Interactive Flask dashboard for displaying predictions and model performance

Technologies Used

Python, Pandas, Flask, OpenWeather API, Sk-Learn, Requests, HTML/CSS

Challenges & Lessons

One key challenge was ensuring data quality and consistency across different cities and time intervals. Data preprocessing, feature selection, and scaling were critical in improving model accuracy. This project strengthened my experience in API integration, regression modeling, and building clean dashboards for communicating results.
