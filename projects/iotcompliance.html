IoT Device Security Review Using NIST Testing Strategies

Evaluated the cybersecurity posture of consumer IoT devices using a NIST-based compliance framework.

Overview

This capstone project focused on evaluating the security readiness of consumer IoT devices using a structured testing framework derived from key NIST (National Institute of Standards and Technology) publications. The goal was to assess and document how well common IoT products meet recommended cybersecurity practices, with an eye toward improving device trustworthiness in home environments.

Images & Screenshots

Key Features

  • Simulated sensor data from multiple IoT devices relevant to short-term rentals
  • MongoDB backend to store time-series data with JSON-like structure
  • Flask dashboard displaying real-time alerts and recent activity summaries
  • Conditional logic for generating warnings (e.g., excessive noise or temperature anomalies)
  • Modular architecture for easily extending to new sensor types or properties

Technologies Used

  • NIST Special Publications: 800-52, 800-63, 8259, and IR 8267
  • Testing methodologies based on standardized compliance frameworks
  • Consumer IoT devices including Amazon Smart Plug, Echo Flex, and Google Nest Mini

    • What I did

  • Developed a comprehensive testing strategy and plan grounded in NIST best practices
  • Assessed IoT devices across six critical areas: Authentication, Public Key Infrastructure (PKI), Cryptography Support, Physical Security, Network Security, and Customer Support
  • Conducted alpha testing using real-world devices to evaluate and refine the framework

    • Challenges & Lessons

    One challenge was translating high-level NIST guidance into specific, actionable tests applicable to consumer-grade devices, which often lack transparency or consistent documentation. Through hands-on testing and collaborative refinement, I gained experience in applying abstract policy frameworks to tangible evaluations, highlighting the gap between compliance recommendations and actual product implementation in the IoT space.
